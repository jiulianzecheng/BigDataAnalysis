'''GSM''' '''(Global System for Mobile Communications''') is a means of telecommunication produced by the European Institute (ETSI) to develop second-generation (2G) digital that is used by cellular networks used by mobile devices such as [[Mobile phone|mobile phones]] and [[Tablet computer|tablets.]] < ref > {{Cite web|title=What Is GSM? The Global System for Mobile Communication Explained|url=https://www.emnify.com/en/resources/gsm|access-date=2021-07-31|website=www.emnify.com|language=en}} < /ref > < ref > {{Cite web|title=Global System for Mobile Communication - an overview {{!}} ScienceDirect Topics|url=https://www.sciencedirect.com/topics/computer-science/global-system-for-mobile-communication|access-date=2021-07-31|website=www.sciencedirect.com}} < /ref >   
 
 
 In 1991, GSM was deployed in [[Finland]]. By the mid-2010s, it became everywhere for mobile communications. < ref > {{Cite web|title=What is GSM (Global System for Mobile communication)?|url=https://searchmobilecomputing.techtarget.com/definition/GSM|access-date=2021-07-31|website=SearchMobileComputing|language=en}} < /ref >
