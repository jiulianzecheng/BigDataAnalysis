 The '''tamaraw''' (''Bubalus mindorensis''), also known as the '''Mindoro dwarf buffalo''', is a small, hoofed [[mammal]]. < ref > {{cite web |url=https://www.itis.gov/servlet/SingleRpt/SingleRpt?search_topic=TSN & search_value=625123# |title=Bubalus mindorensis, Heude, 1888 |author= < !--Staff writer(s); no by-line.-- >  |date=24 July 2014 |website= |publisher=Integrated Taxonomic Information System (ITIS) |accessdate=24 July 2014 }} < /ref >  It is found only on the island of [[Mindoro]] in the [[Philippines]].  The tamaraw was originally more widely spread. Because of human habitation, hunting, and logging, it is present in only a few remote grassy plains and is now an [[endangered species]]. < ref > {{cite web |url=http://www.iucnredlist.org/details/3127/0 |title=Bubalus mindorensis |author=Hedges | display-authors = etal  |date=2013 |website= |publisher=The IUCN Red List of Threatened Species. Version 2014.2.  |accessdate=24 July 2014 }} < /ref >
