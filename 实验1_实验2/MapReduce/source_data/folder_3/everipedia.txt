 == Article development == 
 
 Anyone can contribute to a page by making an account. < ref name=Lee2016 > {{cite news|url=http://www.thedp.com/article/2016/03/wharton-dropout-founds-wikipedia-competitor|title=Wharton dropout creates Wikipedia alternative alongside Rap Genius co-founder|last=Lee|first=Elaine|work=[[The Daily Pennsylvanian]]|date=22 March 2016|url-status=dead|archive-url=https://web.archive.org/web/20171222051947/http://www.thedp.com/article/2016/03/wharton-dropout-founds-wikipedia-competitor|archive-date=22 December 2017|access-date=11 July 2018}} < /ref >  Everipedia allows users to create any page on anything as long as the content is [[citation|cited]] < ref name=James2017/ >  and neutral, < ref name=Bowman2016/ >  even on non-notable topics. < ref name=Christian2017/ >  [[Startup company|Startups]], [[Celebrity|celebrities]] and other contributors are permitted to create their own articles, as long as the content is sourced. < ref name=Olenski2017 > {{cite news|url=https://www.forbes.com/sites/steveolenski/2017/05/09/what-this-startup-can-teach-cmos-about-seo/#20e3434d1b5b|title=What This Startup Can Teach CMOs About SEO|last=Olenski|first=Steve|work=Forbes|date=9 May 2017|url-status=live|archive-url=https://web.archive.org/web/20180110234405/https://www.forbes.com/sites/steveolenski/2017/05/09/what-this-startup-can-teach-cmos-about-seo/#20e3434d1b5b|archive-date=10 January 2018}} < /ref >
