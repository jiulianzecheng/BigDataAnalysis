'''ALIWEB''' (Archie Like Indexing for the Web) is considered the first [[search engine]], as it was the first that allowed you to search for web pages. It was announced in November 1993. < ref > {{cite web |url=https://groups.google.com/forum/?hl=en#!topic/comp.infosystems.www/WQO-hJwFNi8 |title=ANNOUNCEMENT: ALIWEB (Archie-Like Indexing for the WEB) |date=November 30, 1993 |publisher=Google Groups |access-date=26 September 2019 |df=dmy-all }} < /ref >  by the website maker [[Martijn Koster]], and shown in May 1994.
