'''Abby''' [ '''ab'''-ee ] < ref > {{Cite web|title=Definition of Abby {{!}} Dictionary.com|url=https://www.dictionary.com/browse/abby|access-date=2023-04-11|website=www.dictionary.com|language=en}} < /ref >  is a well-liked name for infant females that has existed for hundreds of years. It is a truncated version of the name Abigail, which originates from the [[Hebrew language|Hebrew]] name Avigayil, signifying  " my father's joy "  or  " source of joy. "  This name has an extensive past and has been used in different cultures and languages. < ref > {{Cite web|title=Abby name Meaning and Origin|url=https://babys.name/detail-Abby-Meaning-Origin-Famous-Similar-Sibling.html|access-date=2023-04-11|website=babys.name}} < /ref >  This name was ranked #575 on the US Popular Names in 2021. < ref > {{Cite web|last=OACT|title=Popular Baby Names|url=https://www.ssa.gov/cgi-bin/popularnames.cgi|access-date=2023-04-11|website=www.ssa.gov}} < /ref >
