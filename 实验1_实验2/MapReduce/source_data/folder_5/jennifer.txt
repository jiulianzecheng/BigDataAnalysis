 The name Jennifer is derived from the Welsh name Gwenhwyfar, which means  " white wave "  or  " fair lady. " < ref > {{Cite web|title=Jennifer name Meaning and Origin|url=https://babys.name/detail-Jennifer-Meaning-Origin-Famous-Similar-Sibling.html|access-date=2023-03-23|website=babys.name}} < /ref >  It is a [[female]] name which was first put into use in the 18th century. The name was quite uncommon prior to 1906. Around 1969, however, the name hit number 3 on the [[girl]] name charts, then between 1970 and 1984 '''Jennifer''' became the single most popular name. Since 1990 the name is still common, although considerably less so. Numerous celebrities carry the name '''Jennifer''', for example [[Jennifer Aniston]] and [[Jennifer Jason Leigh]]. The pronunciation of the name is [ '''jen'''-''uh''-fer ]. < ref > {{Cite web|title=Definition of Jennifer {{!}} Dictionary.com|url=https://www.dictionary.com/browse/jennifer|access-date=2023-03-23|website=www.dictionary.com|language=en}} < /ref >  This name was ranked #493 on the US Popular Names in 2021.  < ref > {{Cite web|last=OACT|title=Popular Baby Names|url=https://www.ssa.gov/cgi-bin/popularnames.cgi|access-date=2023-03-23|website=www.ssa.gov}} < /ref >
