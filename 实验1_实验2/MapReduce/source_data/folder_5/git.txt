 '''Git''' ({{IPAc-en|g|Éª|t}}) is a [[distributed revision control]] system. It is a [[computer program]] that helps people create other computer programs together. Git was made to be fast. < ref > {{cite mailing list |mailinglist=linux-kernel |author=Linus Torvalds |url=http://marc.info/?l=linux-kernel & m=111288700902396 |title=Re: Kernel SCM saga.. |date=2005-04-07 |accessdate=2011-11-30}}  " So I'm writing some scripts to try to track things a whole lot faster. " < /ref >  It was created by [[Linus Torvalds]] for use in [[Software development|developing]] the [[Linux kernel]] which he also created. Git's current development is looked after by [[Junio Hamano]]. It is [[Free software|free]] and [[open source]] software released under the [[GNU General Public License]] version 2 [[software license]]. < ref > {{Cite web|title=About - Git|url=https://git-scm.com/about|access-date=2021-05-12|website=git-scm.com}} < /ref >
