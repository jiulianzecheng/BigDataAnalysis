'''''Bolinichthyes''''' is an extinct genus of prehistoric [[Actinopterygii|ray-finned fish]]. 
 
 == Description == 
 It may have resembled the present day [[lanternfish]]. 
 
 == Species == 
 This genus has only one species: 
 
 * ''[[Bolinichthyes longipes]]'' (type)
