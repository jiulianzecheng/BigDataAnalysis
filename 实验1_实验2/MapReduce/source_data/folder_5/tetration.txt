'''Tetration''' is the [[hyperoperation]] which comes after [[exponentiation]]. < ref > {{cite web|url=http://answers.google.com/answers/threadview/id/743129.html |title=Google Answers: addition, multiplication, exponentiation, then ??? |accessdate=2011-11-02}} < /ref >   < big > < math > ^{x}{y} < /math > < /big >  means y exponentiated by itself, (x-1) times. < ref > {{cite web |author=Daniel Geisler |url=http://www.tetration.org/Tetration/index.html |title=tetration.org |publisher=Tetration |accessdate=2011-11-02 |archive-date=2021-05-06 |archive-url=https://web.archive.org/web/20210506210015/http://www.tetration.org/Tetration/index.html |url-status=dead }} < /ref > < ref > {{cite web|url=http://mathworld.wolfram.com/PowerTower.html |title=Power Tower - from Wolfram MathWorld |publisher=Mathworld.wolfram.com |accessdate=2011-11-02}} < /ref > < ref > {{cite web|url=http://ariwatch.com/VS/Algorithms/TheFourthOperation.htm |title=The Fourth Operation |accessdate=2019-09-11}} < /ref >  List of first 4 [[natural number]] hyperoperations, the inverse of tetration is the super root shown in the example 
 #[[Addition]] 
 #: < math > a + n = a + \underbrace{1 + 1 + \cdots + 1}_n < /math > 
 #::{{Mvar|n}} copies of 1 added to {{Mvar|a}}. 
 #[[Multiplication]] 
 #: < math > a \times n = \underbrace{a + a + \cdots + a}_n < /math > 
 #::''{{Mvar|n}}'' copies of ''{{Mvar|a}}'' combined by addition. 
 #[[Exponentiation]] 
 #: < math > a^n = \underbrace{a \times a \times \cdots \times a}_n < /math > 
 #::''{{Mvar|n}}'' copies of ''{{Mvar|a}}'' combined by multiplication. 
 #Tetration 
 #: < math > {^{n}a} = \underbrace{a^{a^{\cdot^{\cdot^{a}}}}}_n < /math > 
 ::''{{Mvar|n}}'' copies of ''{{Mvar|a}}'' combined by exponentiation, right-to-left. 
 
 The above example is read as  " the nth tetration of a " . 
 
 == Examples == 
 *  < math >  ^{2}3 = 3^3 = 27  < /math > 
 *  < math >  ^{3}3 = 3^{({3^3})} = 3^{27} = 7,625,597,484,987  < /math >
