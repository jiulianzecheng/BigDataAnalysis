 == Features == 
 KaTeX claims to be: 
 
 * Fast: It doesn’t need to reload the [[webpage]]. 
 * Print quality: It is based on [[LaTeX]]. 
 * No dependencies: It doesn't need anything else for [[installation (computer programs)|installation]].
