 A '''''siddur''''' ({{lang-he|סדור}}) is a [[Judaism|Jewish]] [[prayer book]]. It has a set order of a list of Jewish prayers and blessings. < ref > [http://www.thefreedictionary.com/siddur Siddur] in The Free Dictionary. < /ref > 
 
 '' " Siddurs " '', which come from a Hebrew root meaning  " order " , < ref > The word '' " [[seder]] " ,'' referring to the ritual Passover meal, comes from the same root. < /ref >  have a very long history and so they can be found in a number of versions in different [[Religious denomination|streams]] in Judaism.
