 '''Subclass''' could mean: 
 
 * [[Subclass (biology)]], a taxonomic rank intermediate between class and superorder 
 * [[Subclass (computer science)]], a class that is derived from another class or classes 
 * [[Subclass (database theory)]], a class which inherits its data members and methods of their superclasses 
 * Subclass ([[set theory]]), a class contained in some other class in the same way that a subset is a set contained in some other set
