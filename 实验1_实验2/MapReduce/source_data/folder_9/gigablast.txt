 '''Gigablast''' is a small web search engine based in [[New Mexico]]. It was started by Matt Wells in 2000 and was part of [[Infoseek]]. 
 
 In July 2013, the website's source code was released under the [[Apache License]]. < ref > {{cite web|title=Gigablast Now an Open Source Search Engine|url=http://www.prnewswire.com/news-releases/gigablast-now-an-open-source-search-engine-217624911.html|work=PR Newswire|accessdate=May 17, 2015|date=July 30, 2013}} < /ref >
