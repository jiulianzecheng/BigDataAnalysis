 '''Kubuntu''' is a [[version]] of the [[free software|free]] [[Linux]] [[distribution]] [[Ubuntu]]. < ref > {{cite web|title=Kubuntu|url=http://distrowatch.com/table.php?distribution=kubuntu|publisher=DistroWatch|accessdate=20 February 2016}} < /ref >  It uses the [[desktop environment]] [[KDE]] instead of [[GNOME]]. Most of the [[software]] used in Kubuntu, is part of the [[KDE]] project, but all other software used in Ubuntu can also be used. < ref > {{cite web|last=Riddell|first=Jonathan|title=Ten years of Kubuntu|url=https://lwn.net/Articles/635880/|publisher=lwn.net|accessdate=20 February 2016}} < /ref > 
 
 Because the only difference of Kubuntu and Ubuntu is the default software, an Ubuntu system can turn into a Kubuntu system, or a Kubuntu system into an Ubuntu system, only by installing some software. 
 
 Some examples of changes in software: 
 *Adept is used instead of Synaptic 
 *Konqueror is used instead of Firefox 
 *Kopete is used instead of Gaim 
 *KTorrent is used instead of Transmission
