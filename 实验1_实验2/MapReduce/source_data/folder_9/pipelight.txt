 '''Pipelight''' is a [[compatibility layer]] that allows [[NPAPI]] plugins for [[Microsoft Windows|Windows]] to be used on [[Linux]]. It is a modified version of [[Wine (software)|Wine]]. [[Silverlight]], [[Adobe Flash Player|Flash Player]], [[Unity (game engine)|Unity Web Player]] and [[Widevine]] run. More plugins, such as [[Adobe Shockwave|Shockwave]], run experimentally. The web browser must be able to use NPAPI plugins. Some web browsers (notably newer versions of [[Google Chrome|Chrome]] and [[Opera (web browser)|Opera]], and Firefox 52 and later < ref > {{cite web | url = https://www.mozilla.org/en-US/firefox | title = Firefox —  Notes (52.0) — Mozilla | date = 2017-03-07 | website = Mozilla.org | publisher = Mozilla | access-date = 2017-03-18}} < /ref > ) are not.
