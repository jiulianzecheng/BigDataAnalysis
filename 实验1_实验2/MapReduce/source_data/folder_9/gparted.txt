 '''GParted''' is a [[computer program]] used for working on  [[Partition (computing)|partitions]]. It can create, delete, move, copy, check and change the size of paretitions. It is also used with the [[file systems]] on the partitions. Thi program is useful for creating space for new [[Operating system|operating systems]]. It works with [[Vista System  &  Data]] partitions. It can also be used to reorganizing disk usage, copy data on hard disks and [[mirroring]] one partition with another ([[Disk image|disk imaging]]).
