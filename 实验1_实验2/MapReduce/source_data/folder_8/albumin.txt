 '''Albumin''' is the name for a number of [[globular protein]]s that can be [[solubility|dissolved]] in [[water]]. < ref > {{cite web |url= http://www.credoreference.com/entry/cuphbe/albumin |title=Albumin |work=Cambridge Dictionary of Human Biology and Evolution |year=2005 |access-date=16 June 2012}} < /ref >  Albumin is found in [[blood serum]]. Albumin is also found in egg white. The albumin index is used to measure the freshness of an egg. < ref > {{cite web |url= http://www.credoreference.com/entry/whdictnutr/albumin_index |title=Albumin index |work=Benders' Dictionary of Nutrition and Food Technology |year=2006 |access-date=16 June 2012}} < /ref >  It measures how far an egg spreads if broken on a flat smooth surface. The older an egg is, the further it spreads.
