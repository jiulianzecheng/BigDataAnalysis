 < code > '''uwu''' < /code > , also used as  < code > '''UwU''' < /code > , is an [[emoticon]] showing cuteness. The  < code > u < /code >  letters represent eyes, while the  < code > w < /code >  represents a mouth. < ref name= " dictionary.com " > {{cite web|url=https://www.dictionary.com/e/slang/uwu/|title=uwu|work=[[Dictionary.com]] Unabridged|publisher=[[Random House]]|access-date=June 4, 2021}} < /ref > < ref name= " dailydot " > {{cite web|last1=Turner|first1=Amy-Mae|title=What does 'uwu' mean?|url=https://www.dailydot.com/unclick/what-does-uwu-mean/|work=[[The Daily Dot]]|access-date=June 4, 2021|date=May 26, 2019}} < /ref >   
 
 == Usage == 
 < code > uwu < /code >  is used to show cuteness, [[:en:Happiness|happiness]], or kindness. < ref name= " dailydot2 " > {{cite web|last1=Turner|first1=Amy-Mae|title=What does 'uwu' mean?|url=https://www.dailydot.com/unclick/what-does-uwu-mean/|work=[[The Daily Dot]]|access-date=June 4, 2021|date=May 26, 2019}} < /ref > < ref name= " rae " > {{Cite web|title=uwu|url=http://www.rae.es/observatorio-de-palabras/uwu|access-date=June 4, 2021|website=Observatorio de palabras de la Real Academia Española|language=es}} < /ref >   Overuse of the emoticon can also have a effect of annoying its receiver. < ref name= " studybreaks " > {{cite web|last1=Srinivasan|first1=Janani Mangai|title=The Origin and Spread of 'UWU' — The Word That Epitomizes Cuteness|url=https://studybreaks.com/tvfilm/the-origin-of-the-word-uwu/|work=[[Study Breaks]]|access-date=June 4, 2021|date=February 22, 2020}} < /ref >  The emoticon has been popular in the [[Furry|furry fandom]]. < ref name= " dailydot2 "  / >   
 
 There's a more surprised variant,  < code > OwO < /code >  (also connected with the furry fandom and the response,  " what's this? " ) < ref name= " dailydot2 "  / >  that may mean cuteness, as well as curiosity and confusion. < ref > {{Cite web|title=People Have Discovered This Weird Furry Meme Is An Actual Word And It's Making Them Scream|url=https://www.buzzfeednews.com/article/krishrach/people-have-discovered-owo-is-an-actual-word-and-its-making|access-date=2021-07-25|website=BuzzFeed News|language=en}} < /ref > 
 
 == History == 
 < code > uwu < /code >  is known to exist back as far back as [[2005]], in an [[anime]] [[fan fiction]]. The origin of the word is not known, with many people believing it to originate in [[Chat room|Internet chat rooms]]. By 2014, the emoticon had spread across the Internet into [[Tumblr]], becoming an [[Subculture|Internet subculture]]. < ref name= " studybreaks2 " > {{cite web|last1=Srinivasan|first1=Janani Mangai|date=February 22, 2020|title=The Origin and Spread of 'UWU' — The Word That Epitomizes Cuteness|url=https://studybreaks.com/tvfilm/the-origin-of-the-word-uwu/|access-date=June 4, 2021|work=[[Study Breaks]]}} < /ref > 
 
 The word  < code > uwu < /code >  is included in the [[Royal Spanish Academy]]'s word observatory,{{Efn|The word observatory is a website owned by the Royal Spanish Academy featuring words that are not included in the Spanish dictionary and haven't been considered for inclusion, but for which people have wanted to know the meaning of, such as technical terms, neologisms, and foreign words. < ref > {{Cite web|last=Morales|first=Manuel|date=2020-10-23|title=La RAE presenta su nueva web, con un 'Observatorio' de palabras que no están en el 'Diccionario'|url=https://elpais.com/cultura/2020-10-23/la-rae-presenta-su-nueva-web-con-un-observatorio-de-palabras-que-no-estan-en-el-diccionario.html|access-date=June 4, 2021|work=[[El País]]|language=es}} < /ref > }} defined as an  " emoticon used to show happiness or tenderness " . < ref name= " rae2 " > {{Cite web|title=uwu|url=http://www.rae.es/observatorio-de-palabras/uwu|access-date=June 4, 2021|website=Observatorio de palabras de la Real Academia Española|language=es}} < /ref > < ref > {{Cite web|last=Mouse|date=October 28, 2020|title=La RAE aclaró las dudas boomers sobre 'uwu'|url=https://www.latercera.com/mouse/la-rae-aclaro-las-dudas-boomers-sobre-uwu/|access-date=June 4, 2021|work=[[La Tercera]]}} < /ref >
