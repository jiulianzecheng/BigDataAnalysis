'''Cholecystitis''' is when the [[gallbladder]] gets [[inflamed]]. < ref name =  " mayo " > {{cite web|url=https://www.mayoclinic.org/diseases-conditions/cholecystitis/symptoms-causes/syc-20364867|title=Cholecystitis - Symptoms and causes|website=Mayo Clinic}} < /ref >  If it is not treated, cholecystitis can become life-threatening, such as if it results in one's gallbladder rupturing. < ref name =  " mayo " / >  Treating cholecystitis often involves removing the gallbladder. < ref name =  " mayo " / >
