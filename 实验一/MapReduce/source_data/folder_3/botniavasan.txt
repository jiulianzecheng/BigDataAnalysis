'''Botniavasan''' is a [[cross-country skiing]] event held around [[Vörå]] in [[Finland]]. The event was first held in 2006. < ref > {{cite web|url=http://komossa.byar.fi/document.asp?id=khxce76nn5e|title=Botniavasan 2006|publisher=Komossa|language=sv|date=2006|access-date=26 July 2013}} < /ref >  The event is held every year in February. The first time the event was first held was in 2006. The main race is 50 kilometers long. The event has been named after [[Vasaloppet]] in [[Sweden]].
