'''Ronaldo''' is a Portuguese < ref > {{Cite web|url=https://www.behindthename.com/name/ronaldo|title=Meaning, origin and history of the name Ronaldo|last=Campbell|first=Mike|website=Behind the Name|access-date=2020-01-26}} < /ref >  given name that is similar to the English [[Ronald]]. It became a common name in all Portuguese-speaking countries. The name is also very common in Italy and Spanish-speaking countries.{{Citation needed|date=July 2023}}.
