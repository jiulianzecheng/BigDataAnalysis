A game currently owned by [[Ubisoft]], < ref > {{Cite web|title=Ubisoft - Brawlhalla|url=https://www.ubisoft.com/en-gb/game/brawlhalla/brawlhalla|access-date=2021-01-15|website=www.ubisoft.com|language=en-gb}} < /ref >  it has [[Super Smash Bros.|super smash bros]] type gameplay and has a player base of 50 million and climbing. < ref > {{Cite web|title=Brawlhalla celebrates 20 million players|url=https://www.shacknews.com/article/109934/brawlhalla-celebrates-20-million-players|access-date=2021-01-15|website=Shacknews|language=en}} < /ref > 
 
 == In game purchases == 
 This game features many in game purchases, a player can earn coins to unlock characters or spend [[money]] to buy them.
