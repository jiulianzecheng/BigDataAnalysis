 '''Aquamarine''' is a [[gemstone]], named for its [[blue-green]] color, which looks like some shallow [[sea]]s. It is a type of [[beryl]]. The color [[aquamarine (color)|aquamarine]] is named for the gemstone. It is considered as a [[birthstone]] of March.
