 '''UNetbootin''' (Universal Netboot Installer) is a [[cross-platform]] utility that can create Linux [[Live USB]] drives and can load a variety of system utilities or install various [[Linux distribution]]s and other [[operating system]]s without a CD. < ref > SourceForge.net, [http://sourceforge.net/apps/trac/unetbootin/wiki/builtin-distributions UNetbootin distributions with built-in support]; retrieved 2012-6-7. < /ref > 
 
 ==Development== 
 UNetbootin is an [[open-source software]] project.  It has 130,000+ weekly downloads. < ref > SourceForge.net, [http://sourceforge.net/directory/system-administration/softwaredist/os:windows/freshness:recently-updated/ Unetbootin download stats]; retrieved 2012-6-7. < /ref > 
 
 For Windows, Mac OS X and Linux systems, UNetbootin may make it possible to do without the [[hard drive]]. < ref > Dörner, Stephan. [https://www.handelsblatt.com/technologie/it-tk/it-internet/kommentar-warum-virenscanner-versagen/6706962.html   " ''So bleibt der PC garantiert virenfrei'' " ] {{Webarchive|url=https://web.archive.org/web/20120826220006/http://www.handelsblatt.com/technologie/it-tk/it-internet/kommentar-warum-virenscanner-versagen/6706962.html |date=2012-08-26 }} (Ensuring the Viruc-free PC), ''Handelsblatt'' (Germany). June 4, 2012; excerpt, ''Für Windows, Mac OS X und Linux gibt es dafür das kostenlose Tool UNetbootin, ... [b]esonderer Vorteil der Live-Distribution: Sie kommt ohne Festplatte aus''; retrieved 2012-6-7. < /ref >
