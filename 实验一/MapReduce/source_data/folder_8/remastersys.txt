 '''Remastersys''' is a [[free software]]/[[open source]] ([[GNU General Public License|GPL]]) script for [[Linux]] [[Linux distribution|distributions]]. Users can: 
 * '''Create a customized Live CD/DVD''' of [[Ubuntu (operating system)|Ubuntu]]. 
 * Backup all the system, including user data to an installable [[Live CD|Live CD/DVD]].
