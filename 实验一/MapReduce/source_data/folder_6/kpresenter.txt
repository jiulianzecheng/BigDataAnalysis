 '''KPresenter''' is a free [[presentation program]] that is part of [[KOffice]], an office suite for the [[KDE]] [[Desktop Environment]]. 
 
 KPresenter's native format is [[XML]], compressed with [[ZIP file format|ZIP]]. KPresenter is also able to load presentations from [[Microsoft PowerPoint]], [[MagicPoint]] and [[OpenOffice.org Impress]] documents. < ref > [http://www.koffice.org/filters/1.6/ Status of Supported File Formats] {{Webarchive|url=https://web.archive.org/web/20080513125754/http://koffice.org/filters/1.6/ |date=2008-05-13 }}.
