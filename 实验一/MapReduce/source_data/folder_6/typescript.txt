 '''TypeScript''' is a [[programming language]] from [[Microsoft]]. It is a type of [[JavaScript]] that adds [[static typing]]. It became public in October 2012 when Microsoft worked on it for two years. < ref > {{cite web|date=1 October 2012|title=Microsoft augments JavaScript for large-scale development|url=http://www.infoworld.com/d/application-development/microsoft-augments-javascript-large-scale-development-203737|access-date=26 April 2015|work=[[InfoWorld]]|publisher=[[International Data Group|IDG]]}} < /ref > < ref > {{cite web|last=Turner|first=Jonathan|date=2 April 2014|title=Announcing TypeScript 1.0|url=http://blogs.msdn.com/b/typescript/archive/2014/04/02/announcing-typescript-1-0.aspx|access-date=26 April 2015|work=TypeScript Language team blog|publisher=Microsoft|archive-date=2 May 2015|archive-url=https://web.archive.org/web/20150502074848/http://blogs.msdn.com/b/typescript/archive/2014/04/02/announcing-typescript-1-0.aspx|url-status=dead}} < /ref > 
 
 TypeScript can be compiled to most versions of [[ECMAScript]] ([[JavaScript]]). < ref > {{Cite web|last=Staff|first=Ars|date=2012-10-03|title=Microsoft TypeScript: the JavaScript we need, or a solution looking for a problem?|url=https://arstechnica.com/information-technology/2012/10/microsoft-typescript-the-javascript-we-need-or-a-solution-looking-for-a-problem/|access-date=2021-08-15|website=Ars Technica|language=en-us}} < /ref >
