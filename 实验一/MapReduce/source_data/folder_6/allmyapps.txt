'''Allmyapps''' is an application store for [[Microsoft Windows]]. The service allows users to install, update and organize over 1,500 PC applications.

 == Features == 
 Allmyapps works with the [[Windows 8]], [[Windows 7]], [[Windows Vista]] and [[Windows XP]] operating systems. 
 
 The service lets users start the installation of applications from its website and from its desktop client. 
 
 The desktop client can detect software already installed on the computer for updating them via the application store and to save the list online as a backup.
