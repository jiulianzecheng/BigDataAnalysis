 '''VirtualBox''' is a general-purpose full virtualizer for x86 hardware, targeted at server, desktop and embedded use. Many Operating Systems (OS) work on VirtualBox, For Example, NextStep OS can be installed on Virtual Box. < ref > {{cite web | url=https://www.osnews.com/story/131254/installing-nextstep-os-openstep-in-virtualbox/ | title=Installing NextStep OS (OpenStep) in VirtualBox – OSnews }} < /ref >   
 
 It is important to note that VirtualBox only provides a virtual (hardware) platform for a virtual computer system. Users of VirtualBox will still need to have their own access to installation source media for the Operating Systems and Software they wish to run using VirtualBox.
