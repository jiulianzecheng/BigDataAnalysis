'''Addlestone''' is a large [[village]] in [[Runnymede (borough)|Runnymede]], [[Surrey]]. It is situated between [[Weybridge]] and [[Chertsey]]. < ref > {{cite web |url=http://www.addlestonehistory.net/ |title=About Addlestone |author= < !--Staff writer(s); no by-line.-- >  |publisher=Addlestone Historical Society |access-date=17 December 2014 |archive-date=16 December 2014 |archive-url=https://web.archive.org/web/20141216190947/http://addlestonehistory.net/ |url-status=dead }} < /ref >  It has a [[railway station]] that has direct links to [[London]] within 1.5 hours. Addlestone is famous for a [[thousand]] year [[oak]] tree named The Crouch Oak. < ref name=EIS > {{cite web |url=http://www.aboutbritain.com/towns/addlestone.asp |title=Addlestone Tourist Information |author= < !--Staff writer(s); no by-line.-- >  |publisher=Excelsior Information Systems Ltd. |access-date=17 December 2014}} < /ref >  There is a legend that Queen [[Elizabeth I]] stopped to picnic under it. < ref name=EIS/ >
