 '''Wynncraft''' is a ''Minecraft'' server, that homes a Minecraft [[massively multiplayer online role-playing game]] (MMORPG) game. Wynncraft was made mostly by Jumla, Salted, and Grian, and it was released in April of 2013. < ref name= " Guinness "  / >  One server owner, Salted, said on [[Twitter]], that over 2.9 million players had joined the server at one point, as of March 2021. < ref name= " Twitter " > {{Cite tweet|number=1372945596991016960|user=CraftedMovie|title=can tweets count as wikipedia references? It says wynncraft has 1.7 million players but we're well above 2.9 million now|archive-url=https://web.archive.org/web/20210319163715/https://twitter.com/CraftedMovie/status/1372945596991016960|archive-date=March 19, 2021|access-date=March 19, 2021|url-status=live}} < /ref > 
 
 == Reception == 
 Wynncraft has been highly perceived, for having complete gameplay, all on Minecraft. Some reviewers praised the game for it's use of Minecrafts vanilla tools, to create a full MMO game. According to [[Guinness World Records]], in July of 2017, it was the biggest MMORPG game built on Minecraft. < ref name= " Guinness " > {{Cite web |date=July 4, 2017 |title=Largest MMORPG built in Minecraft |url=https://www.guinnessworldrecords.com/world-records/481452-largest-mmorpg-built-in-minecraft |website=Guinness World Records}} < /ref >
