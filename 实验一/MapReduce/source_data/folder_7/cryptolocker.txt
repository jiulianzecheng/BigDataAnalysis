'''CryptoLocker''' is a file [[wikt:encrypting|encrypting]] [[ransomware]]. < ref > {{cite web|url=https://arstechnica.com/security/2013/10/youre-infected-if-you-want-to-see-your-data-again-pay-us-300-in-bitcoins/|title=You’re infected—if you want to see your data again, pay us $300 in Bitcoins|date=October 17, 2013|access-date=March 28, 2015|publisher=Arstechnica|last=Goodin|first=Dan}} < /ref >  It first appeared in September 2013. It was [[wikt:transferred|transferred]] using [[E-mail|email attachments]] and [[botnet]]s. The ransomware [[Encryption|encrypts]] most files on the computer (including connected [[wikt:devices|devices]]), using an [[RSA (algorithm)|RSA]] [[wikt:encryption|encryption]].After infection Cryptolocker scan the victim’s folder structure for files matching a set of file extensions, encrypt them and display a message window that demands a ransom in order to decrypt the files. < ref > {{Cite news|url=https://www.theguardian.com/technology/2014/jun/03/cryptolocker-what-you-need-to-know|title=Cryptolocker: what you need to know|last=Hern|first=Alex|date=2014-06-03|work=The Guardian|access-date=2019-03-07|language=en-GB|issn=0261-3077}} < /ref >  Cryptolocker use RSA public-key cryptography.  < ref > {{cite web|date=November 3, 2015|title=RSA-4096 Ransomware Information|url=https://mudra.website/|url-status=dead|archive-url=https://web.archive.org/web/20160522163727/http://www.how-2-remove.com/remove-rsa-4096-virus/|archive-date=May 22, 2016|access-date=March 20, 2016|publisher=Malware Research}} < /ref >  Once it has encrypted the files, it will display a [[Window (computing)|window]] asking for 2 [[Bitcoin]] for the decryption program. < ref > {{cite web|url=http://www.bleepingcomputer.com/virus-removal/cryptolocker-ransomware-information|title=CryptoLocker Ransomware Information Guide and FAQ|publisher=Bleeping Computer|access-date=March 28, 2015|date=October 14, 2013|last=Abrams|first=Lawrence}} < /ref >  It will give the user 72 hours to pay the [[ransom]]. 
 
 If the user does not pay the fee in time, the ransomware will uninstall itself and ask the user to [[download]] it again. It will then ask the user for 10 [[Bitcoin]], instead of 2. < ref > {{cite web|url=http://www.networkworld.com/community/node/84174|title=CryptoLocker crooks charge 10 Bitcoins for second-chance decryption service|publisher=Network World|access-date=March 28, 2015|date=November 3, 2013|archive-date=November 5, 2013|archive-url=https://web.archive.org/web/20131105175934/http://www.networkworld.com/community/node/84174|url-status=dead}} < /ref >  A program to decrypt (fix) files was later made available. < ref > {{cite web|url=https://www.decryptcryptolocker.com/|title=Decrypt CryptoLocker|access-date=2015-03-28|archive-date=2014-08-07|archive-url=https://web.archive.org/web/20140807142550/https://www.decryptcryptolocker.com/|url-status=dead}} < /ref >
