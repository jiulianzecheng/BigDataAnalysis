An '''IRCd''', meaning '''[[Internet Relay Chat]] [[Daemon (computer software)|server program]]''', is a program, [[server (computing)|server software]] that uses the IRC [[protocol (computing)|protocol]].  This allows people to talk to each other on the [[Internet]].  They exchange [[text-based (computing)|textual]] messages in [[real time]]. < ref > {{Cite web |url=http://www.irchelp.org/irchelp/ircd/server-request.html |title=IRC Server Request FAQ |access-date=2009-05-10 |archive-date=2009-04-22 |archive-url=https://web.archive.org/web/20090422113612/http://www.irchelp.org/irchelp/ircd/server-request.html |url-status=dead }} < /ref > 
 
 The server accepts connections from [[IRC client]]s on a group of [[TCP and UDP port|TCP ports]]. When the server is part of an IRC [[network]], it also keeps one or more connections with other servers/[[daemon (computer software)|daemons]]. < ref > [http://www.irc.org/tech_docs/ircnet/faq.html#ircd IRCD FAQ on irc.org] < /ref > 
 
 The word ''ircd'' once referred to only one piece of software.
