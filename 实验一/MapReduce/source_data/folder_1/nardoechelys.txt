'''''Nardoechelys''''' is an [[Extinction|extinct]] genus of prehistoric [[ray-finned fish]].  
 
 == Description == 
 ''Nardoechelys'' was slow like ''[[Urosphen]]''. Unlike the other [[genus]], it was weak, which means that it couldn't attack other [[Fish|fishes]].
