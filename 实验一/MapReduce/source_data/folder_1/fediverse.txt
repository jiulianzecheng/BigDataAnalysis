 The '''fediverse''' is many [[Social network service|social media]] and internet [[software]] that can talk with each other. Federated social media allow people on different social media to like, follow, and comment on each other’s profiles. < ref name= " :0 " > {{Cite web|last=Mir|first=Cindy Cohn and Rory|date=2022-11-16|title=The Fediverse Could Be Awesome (If We Don’t Screw It Up)|url=https://www.eff.org/deeplinks/2022/11/fediverse-could-be-awesome-if-we-dont-screw-it|access-date=2023-07-12|website=Electronic Frontier Foundation|language=en}} < /ref > 
 
 Similar to [[e-mail]], the fediverse is not owned by one person or company. Any person or company can make social media instances on the fediverse. < ref name= " :0 "  / >  And similar to email, these instances can connect and talk with each other. < ref name= " :1 " > {{Cite news|date=2023-07-10|title=Meta plans feed for Threads after users complain|language=en-GB|work=BBC News|url=https://www.bbc.com/news/technology-66155433|access-date=2023-07-12}} < /ref > 
 
 For example, Mastodon gGmbH has the fediverse instance “mastodon.social”, < ref name= " :2 " > {{Cite news|last=Knight|first=Will|title=The Man Behind Mastodon Built It for This Moment|language=en-US|work=Wired|url=https://www.wired.com/story/the-man-behind-mastodon-eugen-rochko-built-it-for-this-moment/|access-date=2023-07-12|issn=1059-1028}} < /ref >  and Gmail has the email address “gmail.com”. < ref > {{Cite web|last=Cline|first=Susan|date=2022-05-26|title=The difference between Gmail, a Google Account and Google Workspace accounts|url=https://www.techrepublic.com/article/the-difference-between-gmail-a-google-account-an-apps-account-and-an-apps-for-business-account/|access-date=2023-07-12|website=TechRepublic|language=en-US}} < /ref >  A person on “mastodon.social” has the fediverse identity “@ username @mastodon.social”, < ref > {{Cite news|last=Taylor|first=Josh|date=2022-11-08|title=What is Mastodon, the social network users are leaving Twitter for? Everything you need to know|language=en-GB|work=The Guardian|url=https://www.theguardian.com/technology/2022/nov/08/mastodon-what-is-it-how-do-i-join-use-find-best-server-list-change-elon-musk-twitter-leaving-social-network-alternative|access-date=2023-07-12|issn=0261-3077}} < /ref >  and a person on “gmail.com” has the email address “ username @gmail.com”. But a “gmail.com” user can send an email to someone who uses a different email provider, like “hotmail.com”. < ref name= " :1 "  / >  Similarly, a “mastodon.social” user can comment on a post by someone who uses a different fediverse instance, < ref > {{Cite news|last=Farrell|first=Henry|date=2022-11-12|title=Analysis {{!}} Musk is wrecking speech moderation on Twitter. There’s an alternative.|language=en-US|work=Washington Post|url=https://www.washingtonpost.com/politics/2022/11/12/musk-is-wrecking-speech-moderation-twitter-theres-an-alternative/|access-date=2023-07-12|issn=0190-8286}} < /ref >  like “[[Mozilla Foundation|mozilla]].social”. < ref > {{Cite web|last=Perez|first=Sarah|date=2023-05-04|title=Mozilla expands its Mastodon investment with beta launch of its own highly moderated server|url=https://techcrunch.com/2023/05/04/mozilla-expands-its-mastodon-investment-with-private-beta-launch-of-its-own-highly-moderated-server/|access-date=2023-07-12|website=TechCrunch|language=en-US}} < /ref > 
 
 == Federated networking services == 
 [[File:Fediverse branches 1.2.png|thumb|Fediverse software, and other federated networks, visualized in a tree figure]] 
 Different fediverse software have different kinds of content. Some are for images, while others are for videos. Fediverse software have multiple instances. Some fediverse software, like Mastodon, < ref name= " :3 " > {{Cite web|title=Servers|url=https://joinmastodon.org/servers|access-date=2023-07-12|website=joinmastodon.org|language=en}} < /ref >  have hundreds of instances that can connect with each other. Usually, people can use federated social media software by opening one of the software’s instances in their [[web browser]] ([[Google Chrome]], [[Mozilla Firefox|Firefox]], [[Microsoft Edge|Edge]]...), by using phone apps, or by using a web app. < ref > {{Cite web|last=Robertson|first=Adi|date=2019-07-12|title=How the biggest decentralized social network is dealing with its Nazi problem|url=https://www.theverge.com/2019/7/12/20691957/mastodon-decentralized-social-network-gab-migration-fediverse-app-blocking|access-date=2023-07-12|website=The Verge|language=en-US}} < /ref >
