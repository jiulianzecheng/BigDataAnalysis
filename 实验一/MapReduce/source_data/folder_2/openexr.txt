 '''OpenEXR''' is a image format that has some similar features to [[JPEG]] and [[PNG]], but has a higher dynamic range and a 32-bit color channel. < ref > {{Cite web|title=Chapter 26. The OpenEXR Image File Format|url=https://developer.nvidia.com/gpugems/gpugems/part-iv-image-processing/chapter-26-openexr-image-file-format|access-date=2021-04-21|website=NVIDIA Developer|language=en}} < /ref >  OpenEXR gives more realism and precision but has huge file sizes. It has been released under the [[BSD licence|BSD license]]. < ref > {{Cite web|title=OpenEXR|url=https://www.openexr.com/license|access-date=2021-04-21|website=www.openexr.com|archive-date=2021-04-21|archive-url=https://web.archive.org/web/20210421172922/https://www.openexr.com/license|url-status=dead}} < /ref >
