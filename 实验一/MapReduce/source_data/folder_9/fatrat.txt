 '''FatRat''' is a [[free and open-source software|free and open-source]] extensible [[download]] manager for Linux. It is built on top of the Trolltech Qt 4 library in C++. Developing of this program was started in November 2006. FatRat provides few advanced features such as speed limiting, segmented downloads. < ref > {{Cite web|title=FatRat Download Manager Extensions|url=http://fatrat.dolezel.info/dev/dolezel-thesis-2011.pdf|url-status=dead|archive-url=https://web.archive.org/web/20120114175416/http://fatrat.dolezel.info/dev/dolezel-thesis-2011.pdf|archive-date=2012-01-14|access-date=2011-11-16}} < /ref >

 == Main features == 
 
 Features and characteristics: < ref name= " pbone "  / > < ref > {{cite web|url=http://fatrat.dolezel.info/|title=Home – FatRat Download Manager|website=fatrat.dolezel.info|access-date=2011-11-16|archive-date=2012-09-03|archive-url=https://www.webcitation.org/6APQvGjQQ?url=http://fatrat.dolezel.info/|url-status=dead}} < /ref > 
 
 * HTTP(S)/FTP downloads 
 * FTP uploads 
 * BitTorrent support 
 * Torrent search 
 * Support for SOCKS5 and HTTP proxies 
 * RapidShare.com FREE downloads 
 * RapidShare.com uploads 
 * RapidShare.com link verification and folder extraction 
 * RapidSafe link decoding 
 * Subtitle search 
 * Scheduler 
 * Remote control via Jabber 
 * Remote control via a web interface 
 * RSS feed and podcasts support
