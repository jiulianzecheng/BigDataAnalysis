 '''Caspio''' is an American software company headquartered in [[Sunnyvale, California|Sunnyvale]], California, with offices in Ukraine, Poland, and the Philippines. Caspio was founded by Frank Zamani in 2000. < ref > {{Cite web|title=Former Autoweb.com founder Frank Zamani is launching an enterprise-level solution. But what does that mean, and how can he succeed?|url=https://www.tmcnet.com/usubmit/2006/05/01/1627716.htm|website=www.tmcnet.com}} < /ref >  The company focuses on  database-centric web applications. < ref > {{cite web|date=January 30, 2018|title=Caspio Surpasses One Million Applications Built on Its Low-Code Platform|url=https://www.businesswire.com/news/home/20180130005545/en/Caspio-Surpasses-Million-Applications-Built-Low-Code-Platform|website=Business Wire}} < /ref > < ref name= " :1 " > {{cite web|date=October 10, 2006|title=Caspio Bridge: Database for Non-Programmers|url=http://www.ciol.com/caspio-bridge-database-programmers/|website=CIOL Bureau}} < /ref > 
 
 == History == 
 Caspio was founded by Frank Zamani in 2000. < ref name= " auto " > {{Cite web|date=October 22, 2008|title=Caspio: Gritty and Profitable In a Tough Market|url=https://readwrite.com/2008/10/22/caspio_gritty_profitable_in_a/|website=ReadWrite}} < /ref >  The company initially focused on simplifying custom cloud applications < ref > {{Cite web|last=Carr|first=David F.|title=Simplifying Database Publishing With Caspio|url=https://www.forbes.com/sites/davidcarr/2010/11/19/simplifying-database-publishing-with-caspio/|website=Forbes}} < /ref > < ref > {{Cite web|date=May 12, 2010|title=Is Caspio The  " Microsoft Access "  Of The Cloud?|url=https://www.informationweek.com/information-management/is-caspio-the-microsoft-access-of-the-cloud-|website=InformationWeek}} < /ref > < ref > {{cite news|last=Smith|first=Jeffery|date=August 5, 2010|title=Gov apps get boost with cloud-based data publishing tool|work=CivSource|url=https://civsourceonline.com/2010/08/05/egov-apps-get-boost-with-cloud-based-database-publishing-tool/|access-date=2022-12-05}} < /ref >  and reducing development time and cost as compared to traditional software development. < ref name= " :1 "  / > 
 
 Caspio released the first version of its platform, Caspio Bridge, in 2001. < ref > {{cite web|date=October 10, 2006|title=Caspio Bridge: Database for non-programmers|url=http://www.ciol.com/caspio-bridge-database-programmers/|website=CIOL Bureau}} < /ref >  In 2014, Caspio released a HIPAA-Compliant Edition of its low-code application development platform. < ref > {{cite web|date=August 1, 2014|title=Caspio HIPAA Enterprise Targets Health Care Industry|url=http://www.eweek.com/it-management/caspio-hipaa-enterprise-targets-health-care-industry|website=eWeek}} < /ref >
