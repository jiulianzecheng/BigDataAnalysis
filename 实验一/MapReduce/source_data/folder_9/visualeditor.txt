 '''VisualEditor''' ('''VE''') is a project developed by the [[Wikimedia Foundation]] together with [[Wikia]]. < ref > {{cite web | url=https://www.theverge.com/2012/6/22/3110130/wikimedia-visual-editor-prototype | title=Wikimedia releases updated prototype for simplified visual editor | date=22 June 2012 }} < /ref >  It is similar to a [[WYSIWYG]] editor, and is available as a [[MediaWiki]] extension. The [[Beta]] version of this extension was enabled on Mediawiki.org and other larger Wikipedia in July 2013 by default, but users could decide to [[opt-out]].
