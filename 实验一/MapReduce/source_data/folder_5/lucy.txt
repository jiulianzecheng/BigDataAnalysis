The name '''Lucy''' [ '''loo'''-see ] < ref > {{Cite web|title=Definition of Lucy {{!}} Dictionary.com|url=https://www.dictionary.com/browse/lucy|access-date=2023-03-31|website=www.dictionary.com|language=en}} < /ref >  is a [[French language|French]] and [[English language|English]] female given name. It is feminine for the male name ''Lucius'' (which means  " as of light " ). < ref > {{Cite web|title=Lucy name Meaning and Origin|url=https://babys.name/detail-Lucy-Meaning-Origin-Famous-Similar-Sibling.html|access-date=2023-03-31|website=babys.name}} < /ref >  There are several celebrities with this name. Fictional characters having the name include Lucy McClane from ''Die Hard''. 
 Lucy is a beautiful, happy, cheerful name. This name was ranked #47 on the US Popular Names in 2021. < ref > {{Cite web|last=OACT|title=Popular Baby Names|url=https://www.ssa.gov/cgi-bin/popularnames.cgi|access-date=2023-03-31|website=www.ssa.gov}} < /ref >
