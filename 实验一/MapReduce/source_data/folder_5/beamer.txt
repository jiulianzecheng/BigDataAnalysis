 '''Beamer''' is an [[open source]] [[LaTeX]] document class for creating [[presentation]] slides. < ref > Mertz, A.,  &  Slough, W. (2005). Beamer by example. The PracTEX Journal. < /ref > < ref > Batts, C. T. (2007). A beamer tutorial in Beamer. The University of North Carolina at Greensboro, Department of Computer Science. < /ref > < ref > Oaks, A. (2009). LATEX Programming: Beamer Presentations. University of California, Berkeley. < /ref >  This [[software]] is mostly used to create slides with mathematical formulas. < ref > This is the biggest difference between [[Microsoft PowerPoint]] and [[LibreOffice]]. < /ref >  It also supports many compile options. By using Beamer, all slides will be generated in [[PDF]] style.
